1) Props are...

Props flow down and are read-only for children.

2) A common way to set default props is...

Example: function Comp({ x = 1 }) { ... }

3) Props are passed...

Data flows down the component tree.

4) Avoid mutating props because...

Mutations can cause unpredictable behavior.

5) You can set default props by...

Example: function Comp({ size = 'md' }) { ... }

6) Props can be...

You can pass data, callbacks, even JSX.

7) Children are accessed via...

Children hold nested elements passed between tags.

8) Spreading props is done with...

Use the spread operator to forward props.

9) Prop drilling refers to...

Consider context to avoid deep drilling.

10) The best practice is to keep components...

Given the same props, they should render the same output.
